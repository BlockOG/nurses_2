nurses_2.widgets.behaviors.themable¶

Themable()

Themable widgets share a color theme.
class nurses_2.widgets.behaviors.themable.Themable¶

Bases: ABC

Themable widgets share a color theme. They must implement a single method, update_theme which repaints the widget whenever the color theme changes.
