A toggle button widget.
class nurses_2.widgets.toggle_button.ToggleButton(*, background_char=' ', label: str = '', callback: typing.Callable[[nurses_2.widgets.behaviors.toggle_button_behavior.ToggleState], None] = <function ToggleButton.<lambda>>, **kwargs)#

Bases: nurses_2.widgets.behaviors.themable.Themable, nurses_2.widgets.behaviors.toggle_button_behavior.ToggleButtonBehavior, nurses_2.widgets.widget.Widget

A toggle button widget. Without a group, a toggle button acts like a checkbox. With a group it behaves like a radio button (only a single button in a group is allowed to be in the “on” state).
