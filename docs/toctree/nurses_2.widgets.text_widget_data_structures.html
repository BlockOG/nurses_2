class nurses_2.widgets.text_widget_data_structures.CanvasView(canvas)¶

Bases: object

A wrapper around a numpy ndarray with a convenient add_text method.

Notes

One-dimensional views will have an extra axis pre-pended to make them two-dimensional. E.g., rows and columns with shape (m,) will be re-shaped to (1, m) so that the add_text row and column parameters make sense.

Parameters:
textstr

Text to add to canvas.

Row or rows to which text is added. This will be passed as-is as the first argument to numpy’s ndarray.__getitem__.

columnint, default: 0

The first column to which text is added.

boldbool, default: False

Whether text is bold.

Whether text is underlined.

strikethroughbool, default: False

Whether text is strikethrough.

Notes

Text is meant to be a single line of text. Text is not wrapped if it is too long, instead index error is raised.
