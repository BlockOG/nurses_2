class nurses_2.widgets.behaviors.focus_behavior.FocusBehavior(ptf_on_focus=True, **kwargs)¶

Bases: object

Focusable widgets can be given “focus” by pressing tab or shift + tab or by clicking on them. When a widget gains focus the on_focus method is called. When a widget loses focus the on_blur method is called.

Parameters:
