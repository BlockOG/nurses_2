scrollwheel_enabledbool, default: True

Allow vertical scrolling with scrollwheel.

arrow_keys_enabledbool, default: True

Allow scrolling with arrow keys.

vertical_proportionfloat, default: 0.0

Vertical scroll position as a proportion of total.

horizontal_proportionfloat, default: 0.0

Horizontal scroll position as a proportion of total.

Raises:
ValueError

If add_widget is called while already containing a child.

Notes

ScrollView accepts only one child and places it in a scrollable viewport. ScrollView’s child’s top and left is set from vertical_proportion and horizontal_proportion, respectively.
