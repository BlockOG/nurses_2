Draggable resize behavior for a widget.
class nurses_2.widgets.behaviors.grab_resize_behavior.GrabResizeBehavior(*, allow_vertical_resize: bool = True, allow_horizontal_resize: bool = True, border_alpha: float = 1.0, border_color: AColor = AColor(red=0, green=0, blue=0, alpha=0), border_size: Size = Size(height=1, width=2), **kwargs)¶

Bases: object

Draggable resize behavior for a widget. Resize a widget by clicking its border and dragging it. Widget dimensions won’t be resized smaller than min_height or min_width.

Parameters:
allow_vertical_resizebool, default: True

Allow vertical resize.

allow_horizontal_resizebool, default: True

Allow horizontal resize.

border_alphafloat, default: 1.0

Transparency of border. This value will be clamped between 0.0 and 1.0.

border_colorAColor, default: TRANSPARENT

Color of border.

border_sizeSize, default: Size(1, 2)

Height and width of horizontal and vertical borders, respectively.

Notes

Borders are added as child widgets. Children added later may overlap or cover the borders. The method pull_border_to_front will correct this.

Attributes:
border_alpha

Background character of the border.

border_color

Color of border.

border_size

Height and width of horizontal and vertical borders, respectively.
