nurses_2.app.run_widget_as_app(widget: type[nurses_2.widgets.widget.Widget], *args, **kwargs)¶

Run a widget as a full-screen app.

Parameters:
widgettype[Widget]

Widget type to be instantiated and run as an app.

*args

Positional arguments for widget instantiation.

**kwargs

Keyword arguments for widget instantiation.

Notes

This is a convenience function provided to reduce boilerplate for the simplest case of an App. The widget type is provided instead of an instance to cover cases where the widget schedules tasks on instantiation as no event loop exists before the App is created.
