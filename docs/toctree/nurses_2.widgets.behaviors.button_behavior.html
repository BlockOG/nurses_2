class nurses_2.widgets.behaviors.button_behavior.ButtonBehavior(*, always_release=False, **kwargs)¶

Bases: object

Button behavior for a widget.

A button has three states: ‘normal’, ‘hover’, and ‘down’.

When a button’s state changes one of the following methods are called: ‘update_normal’, ‘update_hover’, and ‘update_down’.

When a button is released, the on_release method is called.

Parameters:
always_releasebool, default: False

Whether a mouse up event outside the button will trigger it.
