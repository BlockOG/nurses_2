class nurses_2.widgets.window.Window(title='', alpha=1.0, **kwargs)¶

Bases: Themable, FocusBehavior, GrabResizeBehavior, Widget

A movable, resizable window widget.

Parameters:
titlestr, default: “”

Title of window.

alphafloat, default: 1.0

Transparency of window background and border.

Notes

If not given or too small, min_height and min_width will be set large enough so that the border is visible and the titlebar’s label is visible.
